Installation:

博文会自动生成在博客目录下source/_posts\postName.md

新建post之后会生成同名文件夹

post文件夹为同名文件，作为assert folder使用
添加图片：

  1. 复制图片文件到postName/下
  2. 在post中插入代码

执行hexo g 后生成文件在public/下，可以部署到服务器

其他命令：

如果出现如下问题：

Try to install with —no-optional option.
